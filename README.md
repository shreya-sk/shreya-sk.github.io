# Shreya's Digital Diary

A small corner of the internet where I document things I’m building, learning, and exploring.

This site is a collection of my projects, technical notes, research interests, and experiments across software engineering, DevOps, AI, and beyond.

## Built With

* React
* TypeScript
* Vite
* Tailwind CSS
* shadcn/ui

## Development

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
