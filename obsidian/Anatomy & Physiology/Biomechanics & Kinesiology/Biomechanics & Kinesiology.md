# Biomechanics & Kinesiology

---

## Key Definitions

| Term | Definition |
|---|---|
| **Biomechanics** | Scientific study of internal/external forces on the body and how they influence movement |
| **Kinesiology** | Study of human movement (combines anatomy + physiology + biomechanics) |
| **Functional anatomy** | How nervous + skeletal + muscular systems coordinate in real movement |

---

## Anatomical Position Terminology

![[../assets/anatomical-location.jpg]]

| Term | Meaning | Example |
|---|---|---|
| **Superior** | Above | Femur is superior to tibia |
| **Inferior** | Below | Foot is inferior to knee |
| **Anterior** | Front | Quadriceps are anterior |
| **Posterior** | Back | Hamstrings are posterior |
| **Medial** | Closer to midline | Inner knee |
| **Lateral** | Further from midline | Outer knee |
| **Proximal** | Closer to centre/origin | Hip end of femur |
| **Distal** | Further from centre | Knee end of femur |
| **Ipsilateral** | Same side of body | Right hand, right foot |
| **Contralateral** | Opposite side | Right hand, left foot |
| **Supine** | Lying face UP (on spine) | Crunch, bench press |
| **Prone** | Lying face DOWN (on belly) | Plank, superman |

> [!note] Pilates Reference
> Pilates instructors use these terms constantly: supine (most mat work), prone (swimming, back extensions), lateral (side-lying series). Precise anatomical language is essential for clear cueing and documentation.

## Planes of Motion

![[../assets/planes-of-motion.jpg]]

Imagine a pane of glass bisecting the body — movement in that plane cannot break it.

```
┌─────────────────────────────────────────────────────────────┐
│                   PLANES OF MOTION                          │
│                                                             │
│  ┌───────────────┐  ┌───────────────┐  ┌───────────────┐    │
│  │   SAGITTAL    │  │    FRONTAL    │  │  TRANSVERSE   │    │
│  │   (L/R)       │  │   (F/B)       │  │   (T/B)       │    │
│  │   "Arrow"     │  │  "Coronal"    │  │   "Across"    │    │
│  ├───────────────┤  ├───────────────┤  ├───────────────┤    │
│  │Forward/back   │  │Side to side   │  │Rotational     │    │
│  ├───────────────┤  ├───────────────┤  ├───────────────┤    │
│  │Flexion        │  │Abduction      │  │Int rotation   │    │
│  │Extension      │  │Adduction      │  │Ext rotation   │    │
│  │Dorsiflexion   │  │Lat flexion    │  │Pronation      │    │
│  │Plantar flexion│  │Eversion       │  │Supination     │    │
│  │               │  │Inversion      │  │Horiz ab/add   │    │
│  ├───────────────┤  ├───────────────┤  ├───────────────┤    │
│  │Squat, lunge   │  │Side lunge     │  │Cable twist    │    │
│  │Bicep curl     │  │Lateral raise  │  │Golf swing     │    │
│  │Running        │  │Jumping jacks  │  │Throwing       │    │
│  │Bench press    │  │Side shuffle   │  │Russian twist  │    │
│  └───────────────┘  └───────────────┘  └───────────────┘    │
└─────────────────────────────────────────────────────────────┘
```

> [!note] Memory Tricks
> - **Sagittal = sword straight ahead** (arrow direction)
> - **Transverse = across** the body (T for top/bottom split)
> - **Frontal = front** (glass plate splits front/back)

---

### Sagittal Plane Movements (Flexion/Extension)

Top of body → down:

| Joint | Flexion | Extension |
|---|---|---|
| **Cervical** | Chin toward chest | Chin away from chest |
| **Spinal** | Forward bend | Backbend |
| **Shoulder** | Arm raises forward | Arm lowers/extends back |
| **Elbow** | Bending (bicep curl up) | Straightening (bicep curl down) |
| **Wrist** | Palm toward forearm | Backhand away |
| **Hip** | Thigh toward trunk | Femur away from hip |
| **Knee** | Heel toward back of thigh | Heel away |
| **Ankle** | **Dorsiflexion** (toes up, toward shin) | **Plantar flexion** (toes down, pointing) |

**Hyperextension:** Extension BEYOND normal limit → possible injury.

---

### Frontal Plane Movements

**Abduction** = away from midline | **Adduction** = toward midline

> [!note] Memory Trick
> **AB-duction = AWAY from body / AD-duction = ADD to body (bring toward)**

| Joint | Abduction | Adduction |
|---|---|---|
| **Shoulder** | Arm raises out to side | Arm lowers toward body |
| **Hip** | Leg raises out to side | Leg returns to neutral |

**Spine:** Lateral flexion (side bend)
**Cervical:** Lateral flexion (ear to shoulder)

**Foot-specific:**
- **Eversion** — arch collapses inward (weight on inside of foot)
- **Inversion** — weight on outside edge of foot

---

### Transverse Plane Movements

**Rotational movements:**

| Joint | Movement |
|---|---|
| **Cervical** | Rotation left/right (shaking head "no") |
| **Spinal** | Trunk rotation |
| **Shoulder** | Internal rotation (humerus rotates inward) / External rotation (outward) |
| **Forearm** | **Radioulnar supination** (palm up — "soup") / **Pronation** (palm down — "prone to spill") |
| **Hip** | Internal rotation (toes toward midline) / External rotation (toes away) |

> [!note] Memory Tricks
> - **Soup = Supination** (palm up holding bowl)
> - **Prone to spill = Pronation** (palm down)

**Horizontal abduction/adduction** — at shoulder/hip when joint is flexed 90°:
- Horizontal abduction = arm moves from front to side
- Horizontal adduction = arm moves from side to front
- Examples: chest fly, chest press — start in horizontal abduction, move to horizontal adduction

---

### Foot — Multiplanar Movement

The foot rarely performs single-plane movements in isolation.

**Foot pronation** = multiplanar combination:
- Eversion (frontal)
- Dorsiflexion (sagittal)
- Forefoot abduction (transverse)

**Foot supination** = multiplanar combination:
- Inversion (frontal)
- Plantar flexion (sagittal)
- Forefoot adduction (transverse)

**Walking/running:** Foot moves between pronation (force reduction) and supination (force production).

### Scapular Movements

![[../assets/scapular-movement.jpg]]

Scapular movements are called **translations** (not rotations around fixed axis).

| Movement | Description |
|---|---|
| **Retraction** | Shoulder blades together |
| **Protraction** | Shoulder blades apart |
| **Elevation** | Upward (toward ears) |
| **Depression** | Downward |

### Force Couple Relationship

Multiple muscles acting on same joint to produce ONE coordinated movement — even if forces are in different directions.

**Car steering analogy:** Left hand pulls down, right hand pushes up → car turns left.

**Body example — upward rotation of scapula (overhead reach):**
- Serratus anterior
- Upper trapezius
- Lower trapezius

All three pull scapula in different directions, but coordinated = smooth upward movement.

## Muscle Actions

### The Full Spectrum

Bicep curl illustrates all three:

| Action | What happens | Force vs Load | Curl phase | Synonym | Example |
|---|---|---|---|---|---|
| **Concentric** | Muscle SHORTENS | Force > load | Lifting up | Acceleration | Lifting weight |
| **Isometric** | Muscle maintains LENGTH | Force = load | Holding top | Stabilisation | Pause during lift |
| **Eccentric** | Muscle LENGTHENS (returning to resting length, not stretching) | Force < load | Lowering down | Deceleration | Lowering the weight ("negative") |

> [!note] Key Programming Principle
> Most training overemphasises CONCENTRIC. Train all three:
> - Maximises strength potential
> - Maintains joint range of motion
> - Prevents injury

> [!note] Pilates Reference — Eccentric Emphasis
> Pilates naturally emphasises the full spectrum — slow, controlled eccentric phases (lowering the leg back down, returning the carriage on the reformer) are where much of the strength benefit lives. Reformer springs deliver resistance in all three phases simultaneously.

### Example — Lower Leg Muscle Actions

**Tibialis Anterior:**
- **Concentric:** accelerates dorsiflexion + inversion
- **Eccentric:** decelerates plantar flexion + eversion
- **Isometric:** stabilises foot arch

**Posterior Tibialis:**
- **Concentric:** accelerates plantar flexion + inversion
- **Eccentric:** decelerates dorsiflexion + eversion
- **Isometric:** stabilises foot arch

## Muscle Roles

| Role | Function | Example (elbow flexion) |
|---|---|---|
| **Agonist** (prime mover) | Primary muscle doing work | Biceps brachii |
| **Antagonist** | Opposes agonist | Triceps brachii |
| **Synergist** | Assists agonist | Brachialis |
| **Stabiliser** | Contracts isometrically to protect joint | Rotator cuff (for shoulder during curl) |

### Muscle Role Examples

| Exercise | Agonist | Synergist | Stabiliser | Antagonist |
|---|---|---|---|---|
| Chest press | Pectoralis major | Anterior deltoid, triceps | Rotator cuff | Posterior deltoid |
| Overhead press | Deltoid | Triceps | Rotator cuff | Latissimus dorsi |
| Row | Latissimus dorsi | Posterior deltoid, biceps | Rotator cuff | Pectoralis major |
| Squat | Glute max, quadriceps | Hamstrings | Transversus abdominis | Psoas (hip flexor) |

**A muscle's role changes based on the movement.** Example — Infraspinatus:
- **Agonist** for shoulder external rotation
- **Synergist** when reaching overhead
- **Stabiliser** during other shoulder movements

### Feed Forward Activation

Some muscles (stabilisers especially) contract AUTOMATICALLY in anticipation of movement.

**Example:** Core engages milliseconds before arm movement, protecting the spine preemptively. See [[../Muscular System/Muscular System#Feed Forward Activation|Muscular System]].

## Levers in the Human Body

![[../assets/lever-examples.jpg]]

**Lever = rigid bar that pivots around stationary fulcrum**

In the body:
- **Fulcrum** = joint axis
- **Lever** = bone
- **Effort** = muscle creating motion
- **Resistance** = weight (body part or external object)

```
┌───────────────────────────────────────────────────┐
│              LEVER CLASSES                        │
│                                                   │
│  CLASS 1: E ─── F ─── R                           │
│  (fulcrum in middle — seesaw)                     │
│  Body example: Nodding head                       │
│  (top of spine = fulcrum)                         │
│                                                   │
│  CLASS 2: F ─── R ─── E                           │
│  (resistance in middle — wheelbarrow)             │
│  Body example: Calf raise                         │
│  (ball of foot = fulcrum, body weight = R,        │
│   calf muscles = E)                               │
│                                                   │
│  CLASS 3: F ─── E ─── R                           │
│  (effort in middle — tweezers)                    │
│  Body example: Bicep curl                         │
│  (elbow = fulcrum, biceps = E,                    │
│   weight in hand = R)                             │
│  ★ MOST COMMON in human body                      │
│  (Effort travels shorter distance than resistance │
│   and must be GREATER than resistance)            │
│                                                   │
└───────────────────────────────────────────────────┘
```

> [!note] Pilates Reference — Spring Mechanics
> Reformer springs act as resistance (R) in the lever system. Foot bar placement, spring attachment position, and carriage distance all change the mechanical advantage — this is why the "same" exercise feels harder with straps longer vs shorter, or with springs attached proximally vs distally.

## Open vs Closed Chain Exercises

Classification by what the **distal segments** (hands/feet) are doing.

| Feature | Closed Chain | Open Chain |
|---|---|---|
| Distal segment | FIXED on stable surface | FREE to move in space |
| Joints involved | Multiple | Usually isolated/fewer |
| Muscle recruitment | Multiple groups | Individual muscles |
| Weight-bearing | Usually YES | Usually no |
| Examples | Push-up, squat, lunge, pull-up | Bicep curl, bench press, leg extension, lat pulldown |

**Same muscle group, different chain type:**
- **Pull-up** = closed chain (hands fixed on bar)
- **Lat pulldown** = open chain (bar moves through space)

**Push-up insight:** Hands fixed, body moves → closed chain.

> [!note] Pilates Reference — Open vs Closed Chain
> Reformer footwork (feet on footbar, carriage moves) = **closed chain** — high functional carryover to standing and gait.
> Leg springs (legs free in space) = **open chain** — great for hip isolation, muscle targeting.
> Most Pilates mat exercises are closed chain (feet on mat, body moves) — this is why Pilates has excellent functional carryover to daily life.

## Applied Biomechanics — Practical Programming

**1. Clear anatomical communication:** Use precise terms (medial, lateral, anterior, posterior) for effective cueing.

**2. Balance all three planes:**
- Most clients over-train the sagittal plane
- Include frontal (side lunges, lateral raises)
- Include transverse (rotations, twists)
- Most daily/sport movements are multiplanar

**3. Train the full muscle action spectrum:**
- Coach eccentric deliberately ("control the lowering")
- Use tempo prescription (e.g. 3 sec eccentric)
- Include isometric holds for stability

**4. Leverage lever mechanics:**
- Hand/foot placement changes mechanical advantage
- Grip width changes muscle loading
- Stance adjustments = different outcome from same exercise

**5. Multi-planar training** = real-world-ready clients.

## Applied Biomechanics — Squat Analysis

A full kinetic chain integration example showing all body systems in a single movement:

### Feet and Ankles

**Muscles:** Tibialis anterior, gastrocnemius, soleus, peroneals.

**Function:** Base of support. Ankles move into dorsiflexion (sagittal plane) during descent. Tibialis anterior controls dorsiflexion; peroneals maintain lateral stability. Gastrocnemius and soleus work eccentrically on descent, concentrically on ascent.

### Knees

**Muscles:** Quadriceps (rectus femoris, vastus lateralis, vastus medialis, vastus intermedius), hamstrings (biceps femoris, semitendinosus, semimembranosus), gastrocnemius.

**Function:** Knee flexes during descent, extends during ascent (sagittal plane). Quadriceps: eccentric down, concentric up — primary agonist. Hamstrings: synergists + stabilisers (prevent excessive tibial forward translation). Proper knee tracking over second toe minimises joint stress.

### Hips

**Muscles:** Gluteus maximus (primary agonist), hamstrings (synergists), gluteus medius/minimus (stabilisers), hip flexors/iliopsoas (antagonists during hip extension).

**Function:** Primary power driver. Descent: hips flex, glute max works eccentrically. Ascent: glute max concentric → hip extension. Gluteus medius stabilises pelvis in frontal plane → prevents hip drop or valgus knee collapse.

### Spine

**Muscles:** Erector spinae, multifidus, quadratus lumborum, transversus abdominis, rectus abdominis.

**Function:** Spine must maintain neutral position throughout. Excessive flexion stresses intervertebral discs. Erector spinae and multifidus work isometrically to resist spinal flexion. TVA creates intra-abdominal pressure stabilising the lumbar spine. This coordinated isometric "core canister" contraction = textbook stabiliser muscle function.

### Shoulders

**Muscles:** Trapezius (upper, middle, lower), deltoids, rhomboids, rotator cuff (infraspinatus, supraspinatus, subscapularis, teres minor).

**Function:** Not primary movers, but critical stabilisers. Middle/lower trapezius + rhomboids maintain scapular retraction/depression — keeping chest open, thoracic spine extended, preventing upper back rounding. Rotator cuff stabilises the glenohumeral joint.

In bodyweight squat: arms in shoulder flexion (sagittal plane), acting as counterbalance — anterior deltoid maintains this isometrically.

> [!note] Kinetic Chain in Action
> From a kinetic chain perspective: protracted scapulae or internally rotated humeri → thoracic flexion → increased lumbar stress. This perfectly illustrates how dysfunction in one kinetic chain component directly affects all others throughout a movement. See [[../Kinetic Chain/Kinetic Chain|Kinetic Chain]].
